  • 万能输入法

    万能输入法 是一款iOS输入法App(目前开源了部分代码),开发者付出了巨大心血在做的一个App。
    也是我本人在iOS技术路上一直努力,不断积累吸收的他人和对比国内外其他键盘App后,开发的一个功能多又有特色的中文输入法。
    支持全/九键盘拼音输入、笔画五笔手写输入、英文补全输入,还支持表情、符号、动图、添加 自定表情、换肤、换字体、自定义主题样式等诸多自定功能。
    这是一个集丰富多彩的功能于一身的、各种强大技巧于一体的iOS输入法。 Github

  • 我的浏览器

    我的浏览器 是一款基于WKWebView封装的浏览网页的浏览器,主要功能有多窗口,收藏夹,日夜间模式,无图模式,广告拦截,支持网页弹窗等功能。
    目前这个App投入精力少,目前已开源,以后作者有时间就会更新优化。希望访者看来后可能加入这个项目维护中来出一份开源力量。 Github

  • MarkDown记事本

    MarkDown记事本 是一款Markdown记事本兼文档编辑器。支持各样格式文档及图片的导入,支持多样化的皮肤设置,PDF文档导出,PD快速预览,以及分享PDF文档。
    同时支持个性化的字体（细体、草书、行书、隶书、幼圆、雅痞、雅黑等字体）。。

    MarkDown记事本

  • 美图王

    美图王 是一款Markdown文档编辑器。支持多样化的皮肤设置,PDF文档导出,PD快速预览F,以及分享PDF文档。
    同时支持个性化的字体（细体、草书、行书、隶书、幼圆、雅痞、雅黑等字体）。。

  • 关于视频教程

    Wodedata播客站 是笔者今年搭建起来的用于itunes播客更新的站点。
    目前发布了3个视频教程,经过了反复的压缩处理后,还是发现占用笔者阿里云服务器流量太大了,用这个东西在自己服务上发布视频,并且完全免费,这个钱烧不起。
